【グラフィック合成機でプレビュー画像が表示されない方へ】

画像が表示されない最も有力な可能性として、お使いのWindowsに
「Gdiplus.dll」というライブラリが存在していない可能性があります
このDLLは、通常のWindows 7以降には標準で搭載されていますが、
軽量化された環境や仮想PC、一部の古いOSでは含まれていない場合があります。

ただし「Gdiplus.dll」は、再配布が難しいライブラリであり、
通常は意図的に手に入れることが難しいファイルです。
以下に、一般的に知られている入手方法を記載します。


【対処方法1】
「.NET Framework 1.1 再頒布パッケージ」をインストールする。  

※このバージョン1.1のパッケージには
　Windows XPでGdiplus.dllが未搭載だった可能性に備えるため、
　過去に Gdiplus.dll が含まれていたそうです。

「.NET Framework 1.1 再頒布パッケージ」公式URL：  
https://www.microsoft.com/ja-jp/download/details.aspx?id=26

※2025/04/05時点で404になっている可能性があります。
　その場合は「.NET Framework 1.1 再頒布パッケージ ダウンロード」などの
　検索ワードで探してみてください。



【対処方法2】
他のPCなどから Gdiplus.dll を探し、以下のいずれかに配置する：

- GraphicMaker.exe と同じフォルダ
- または Windows の `System32` や `SysWOW64` フォルダ（管理者権限が必要です）

※DLLファイルの個別配布サイトは悪意あるものも含まれる可能性があるため、  
　あくまで「自己責任」でご利用ください。



【対処方法3】
Windowsを再インストールする。  
かなり手間ではありますが、標準構成に戻す手段のひとつです。
